04——js 比较和逻辑运算符（关系运算符）

1. 比较和逻辑运算符用于测试 true 和 false

2. 比较运算符 ：

   在逻辑语句中使用，以测定变量或值是否相等

   ```javascript
   ==,===（值和类型全相等，返回true）,!=,>,<,>=,<=,!==（值和类型全不等）
   ```

3. 逻辑运算符：

   用于测定变量或值之间的逻辑

   &&，||，！

4. 运算符优先级：

   小括号 > 算术运算符 > 比较运算符 > 逻辑运算符 > 赋值运算符

5. 条件运算符：

   x == 1 ? "是" : "否"；

   x是否等于1 ，等于则返回“是”，否则 返回“否”
